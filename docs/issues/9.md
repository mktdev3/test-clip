# Issue #9: インデックス作成スクリプトの実装

## 概要
images/フォルダ内の画像をすべてエンコードしてインデックスを作成するスクリプトを実装する。

## 実装例
```python
import os
from src.clip_model import CLIPModel
from src.image_encoder import ImageEncoder
from src.vector_store import VectorStore

def build_index(image_dir, vector_dir, metadata_dir):
    clip = CLIPModel("rinna/japanese-clip-vit-b-16")
    encoder = ImageEncoder(clip)
    store = VectorStore(vector_dir, metadata_dir)
    
    vectors = []
    metadata = []
    
    for filename in os.listdir(image_dir):
        if filename.endswith(('.jpg', '.png')):
            path = os.path.join(image_dir, filename)
            vector = encoder.encode_image(path)
            vectors.append(vector)
            metadata.append({'path': path, 'filename': filename})
    
    store.save(vectors, metadata)
```

- data/vectors/vectors.npy が作成される
